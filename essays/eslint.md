## Before the standard
Before using coding standards, I had been formatting my code to personal preference, adjusting it visually to how I felt it would be easier to comprehend. One such unorthodox adjustment I enacted were the curly braces for functions being placed on a seperate line fo the function name. When I first learned coding, I found it hard to keep track of the opening and closing braces of the functions/classes. As the instructors had their beginning curly braces opened on the same line of teh function's creation, it differed greatly aesthetically from my code. This caused a great rift in how a person viewed my code compared to everyone else which slowed down the process of grading.
